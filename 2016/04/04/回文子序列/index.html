回文子序列

  • 简单算法 动态规划 递归

在此记录一道腾讯笔试的编程题。这题乍一看觉得是用动态规划，但是并没有做出来，最后是用递归解的。递归的思想更容易但是时间复杂度很高，所以我估计这题最后提交上去会超时ORZ。接下来回顾这道题，并再举别的和此题类似的例子~~~

题目描述

给一个字符串，找出它的最长的回文子序列的长度。例如，如果给定的序列是“BBABCBCAB”，则输出应该是7，“BABCBAB”是在它的最长回文子序列。 “BBBBB”和“BBCBB”也都是该字符串的回文子序列，但不是最长的。

当时的题目写的不是这样直白，好像是说一个字符串可以删除任意个字符，求删除后字符串是回文字符串的最大长度。当时就觉着直接求所有子串，然后从长到短来依次判断是否是回文串，如果是则直接return长度。但是这种穷举法显然太弱了。接下来先用递归方法来求解吧（虽然时间复杂度高，但是代码好写…）。

递归解法

思路

给定一个字符串，从两边开始对比（当然从两边，因为谁也不嫌长，而且是子序列，不要求连在一起），假设此时左右下标分别为i和j。

  • 如果str[i]==str[j]，则len+1，然后递归str(i+1,j-1)。
  • 如果str[i]!=str[j]，则分别递归str(i+1,j)和str(i,j-1)，取两者的较大值。

代码

public static int longest(String s) {
    if(s.length() <= 1) return s.length();
    if(s.charAt(0) == s.charAt(s.length()-1)){
        return longest(s.substring(1, s.length()-1))+2;
    }
    int a = longest(s.substring(1, s.length()));
    int b = longest(s.substring(0, s.length()-1));
    return Math.max(a, b);
}

DP解法

思路

dp[i][j]存放字符串str中从CharAt(i)到CharAt(j)的最长回文子序列长度。

  • 如果str[i]==str[j]，dp[i][j]=dp[i+1][j-1]+1。
  • 如果str[i]!=str[j]，dp[i][j]=max(dp[i+1][j],dp[i][j-1])。

然而，递归表达式写出来了，还要考虑初始条件及入口。

  • 初始条件： dp[i][i]=1。
  • 入口：由前面给出的DP表达式可以看出来，在求解dp[i][j]时，要已经得到dp[i+1][j-1]，dp[i+1][j]，dp[i][j-1]这三个变量。

从矩阵的角度来看这个dp二维数组。首先，肯定是j>i，即我们用的是该矩阵的右上半角。其次，初始化时，dp[i][i]=1表示该矩阵的对角线上的数字都为1。再看入口条件，dp[i+1][j-1]，dp[i+1][j]，dp[i][j-1]都在dp[i][j]的右下角。分析完了，可以感受到如何构造这个dp数组了吧~

由下列矩阵可知，在给问好处赋值的时候，X和+处应该已经有值。因此，我们应该从对角线开始，依次向右上角给矩阵赋值。

下图显示地描述了这个过程（假设此时str=”abca”）

代码

public static int longest(String s) {
    int n = s.length();
    int[][] dp = new int[n][n];
    int tmp;
    char[] str = s.toCharArray();
    //初始化
    for(int i=0; i<=n-1; i++){
        dp[i][i] = 1;
    } 

    for(int i=1; i<=n-1; i++){
        tmp = 0;
        //考虑 str[j, j+i] 由j到j+i的串
        for(int j=0; j+i<=n-1; j++){
            //如果首尾相同
            if(str[j] == str[j+i]){
                tmp = dp[j+1][j+i-1] + 2;
            }else{
                tmp = Math.max(dp[j+1][j+i],dp[j][j+i-1]);
            }
            dp[j][j+i] = tmp;
        }
    }
    //返回串 str[0][n-1] 的结果
    return dp[0][n-1];
}

类似解法——最长公共子序列

题目描述

两个字符串Str1和Str2，求两个字符串的公共子序列（顺序不可打乱）的长度。比如”abcdef”和”ccaf”的最长公共子序列为”cf”，长度为2。

思路

我们可以感受到这题可以用dp[i][j]保存str1(0,i)和str2(0,j)的最长公共子序列。(此处不写s.toCharArray或者charAt，意思就是这样)

  • 当str1[i]==str2[j]时，dp[i][j]=dp[i-1][j-1]+1
  • 当str1[i]!=str2[j]时，dp[i][j]=max(dp[i-1][j],dp[i][j-1])
    在矩阵上表示为从矩阵的左上角求解到右下角。一般来说我们需要初始化dp[0][]与dp[][0],但也可以加一行一列赋值为0再直接进入dp过程。
